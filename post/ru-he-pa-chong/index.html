如何爬虫笔记

2020-02-06 / 1 min read

😊😛🤣😚🤓😎🙃
爬虫四步骤：
1.获取数据
2.解析数据
3.提取数据
4.存储数据

1.获取数据：

import requests
#引入requests库

res = requests.get('URL')
#requests.get是在调用requests库中的get()方法，它向服务器发送了一个请求，括号里的参数是你需要的数据所在的网址，然后服务器对请求作出了响应。
#我们把这个响应返回的结果Response赋值在变量res上。

1.0 Response 对象常用属性

属性 response.status_code response.content response.text response.encoding
作用 检查请求是否成功 把response对象转换成二进制数据 把response对象转换成字符串数据 定义response对象的编码

e.g.:

import requests 
res = requests.get('https://res.pandateacher.com/2018-12-18-10-43-07.png') 
print(res.status_code)
#打印变量res的响应状态码，以检查请求是否成功
